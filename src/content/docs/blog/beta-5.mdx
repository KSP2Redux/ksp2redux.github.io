# v0.2.3 Beta 5
We have just released the 5th beta of KSP2 Redux, which includes an upgrade of the game's Unity version from 6000.3.10f1 to 6000.4.1f1, an early release of the modding SDK, an experimental KSP1 part mod loader, first few SM+ size (1.875m) parts, support for Discord rich presence, custom flags, improved SAS and flight controls, and as always, a ton of bugfixes.

**You can download the new Updater application on the [Updater GitHub page](https://github.com/KSP2Redux/Updater/releases/tag/updater-v0.0.2.6).** This application will now be the primary way of installing, updating and uninstalling KSP2 Redux. You will need a stock install of KSP2 0.2.2 for the initial install (meaning you __CANNOT__ upgrade from beta 4), but this won't be the case for any future updates, those will just be installed on top of beta 5.

You can find the full changelog on the [Redux GitHub page](https://github.com/KSP2Redux/Redux/releases/tag/v0.2.3.0-beta%2B7b1b3356) or on Discord: https://discord.com/channels/1078696971088433153/1444156522127954064/1497424090955251854.
