## What is KSP2 Redux?

KSP2 Redux is a fan project built on top of Kerbal Space Program 2. The current work is mostly focused on:

- **Performance work:** Rendering, simulation, and memory improvements
- **Bug fixing:** Flight issues, UI problems, mission logic, map view behavior, and other stock problems
- **Roadmap systems:** Heat, vessel ISRU, colony-related groundwork, and later planned features
- **Modding support:** A better workflow for Redux-native mod development

Redux is a volunteer project. Work moves when people have time to finish something useful.

## Current status

Redux is currently in **Stage 0: Foundation**. That means the public work is still centered on making the base game less fragile and easier to build on.

Current Foundation work includes:

- Performance optimizations and stability work
- Bug fixing across flight, UI, missions, and map view
- Debug tools for players and modders
- Modding SDK improvements
- Vessel ISRU, heat, and tech tree work that later stages depend on

## Important Notes

- **Beta Software:** Redux is currently in beta. Some features may be incomplete or buggy.
- **No Official Affiliation:** Redux has no connection to the former KSP2 developers or Private Division.
- **Volunteer Project:** Redux is developed entirely by volunteers in their free time.
- **Separate Installation:** Redux installs separately from vanilla KSP2 - your original game remains untouched.

## Support

Need help or found a bug?

- **Discord:** Ask questions in **#🟡redux-chat** or report bugs in **#🟡redux-bugs**
